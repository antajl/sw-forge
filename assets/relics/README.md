# Relic icons (local)

Place PNG files here for the **Runes → Table → Relics** category column.

Naming (first match wins):

1. `{typeId}.png` — SWEX relic `type` (e.g. `6.png` = Resolute, `12.png` = Timeless)
2. `{category-slug}.png` — lower case, spaces → hyphens (e.g. `resolute.png`)
3. `default.png` — fallback when nothing else matches

Recommended size: **28×28** or **56×56** (shown at 24px). Transparent background works best.

SWARFARM does not host per-relic artwork; artifact sprites are not used for relics anymore.
