Guide

Data flow: Export JSON from SWEX and load it directly. SWOP CSV is not required.

Database slots: Open top-right Settings to store up to 4 JSON databases. Use Swap to switch active DB instantly.

Dashboard cards: Total, Keep, Sell, Grind, Gem, Finish, Upgrade and Reapp are recalculated for the selected stage.

Global Min Lvl: Header filter applies to all dashboard charts and Rune Table at the same time.

Role Distribution: Shows count and average efficiency per role. Custom roles appear automatically after saving rules.

Set Distribution: Bars show set count and average efficiency to identify high-value sets.

Slot Distribution: For each slot, you see count and top main stat patterns.

Rune Table basics: Search, filter by verdict/role/grade, sort by columns, and inspect innate + substats.

Target column: Appears for Grind/Gem views. Grind shows upgradable stat; Gem shows suggested replacement target.

Rune Summary panel: Right side summary shows counts and average efficiency by verdict and role for filtered rows.

Rune Rules / Thresholds: Base thresholds, High Roll thresholds, Duo Roll thresholds and partner coefficient are edited in one place.

Role Filters: Add custom roles with +Add role, tune Include/Exclude rules, and remove roles (at least one role is kept).

Reapp Rules: Configure allowed sets, innate stats, main stats by slot and max efficiency for Reapp verdict.

Changelog: Keep project notes directly in-app using the Changelog tab.
