Guide

Data flow: Export JSON from SWEX and load it directly. SWOP CSV is not required.

Database slots: Your first SWEX file opens as Data 1. In App Settings you can store up to four exports, see wizard name/level from the JSON, and Swap the active profile.

Dashboard cards: Total, Keep, Sell, Grind, Gem, Finish, Upgrade and Reapp are recalculated for the selected stage.

Account progression suggestion (Dashboard): Early / Mid / Late controls rule strictness. The suggestion uses all runes at +9+ in your export and ignores Min Lvl; use Apply suggestion or pick manually.

How the score is computed: All three terms use Mid rules only (thresholds for power, Mid Keep verdicts, meta among Keep). On-screen preset does not change Combined score. Formula: 40% / 30% / 30%. Suggested stage uses stricter boundaries than the original spreadsheet so "Early" means weak progression: combined score below 43 → Early, below 52 → Mid, else Late. SWEX has no efficiency field — uncapped SWOP-style % is used here; Rune Table eff stays capped 0–100%.

Global Min Lvl: On the Dashboard, the scope bar filter applies to summary cards, charts, Rune Table, and Action List only—not to the progression suggestion block.

Role Distribution: Shows count and average efficiency per role. Custom roles appear automatically after saving rules.

Set Distribution: Bars show set count and average efficiency to identify high-value sets.

Slot Distribution: For each slot, you see count and top main stat patterns.

Rune Table basics: Search, filter by verdict/role/grade, sort by columns, and inspect innate + substats.

Action List: Only runes that need Upgrade, Finish, Gem, Grind, or Reapp; Target explains the next step. Export CSV for offline use.

Target column: In Rune Table it appears when filtering Gem/Grind. In Action List it is always visible (grind goals, innate reroll, +9/+12/Reapp).

Rune Summary panel: Right side summary shows counts and average efficiency by verdict and role for filtered rows.

Rune Rules / Thresholds: Base thresholds, High Roll thresholds, Duo Roll thresholds and partner coefficient are edited in one place.

Role Filters: Add custom roles with +Add role, tune Include/Exclude rules, and remove roles (at least one role is kept).

Reapp Rules: Configure allowed sets, innate stats, main stats by slot and max efficiency for Reapp verdict.

Changelog: Keep project notes directly in-app using the Changelog tab.
